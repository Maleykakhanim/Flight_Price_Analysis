# ✈️ Flight Ticket Price Analysis

## 🧾 Project Title  
**Flight Ticket Price Analysis**

---

## 🎯 Project Objective  
This project aims to analyze airline flight data and identify the key factors that influence ticket prices.

The goal is to explore how variables such as airline, class, departure time, and booking timing affect ticket pricing, and to extract meaningful insights for better decision-making.

---

## ☁️ Environment & Files  
- ☁️ Developed using Google Colab  
- 💻 Notebook file: `Airlines_flights.ipynb`  
- 📂 Dataset: `airlines_flights_data.csv`  

---

## 🔍 Features & Analysis  

- Dataset overview and statistical summary  
- Airline frequency analysis  
- Departure and arrival time distribution  
- Source and destination city comparison  
- Price analysis based on:
  - Airline  
  - Departure and arrival time  
  - Cities  
  - Class (Economy vs Business)  
  - Days left before departure  
  - Number of stops  

- Correlation analysis  
- Boxplot visualization for price distribution  
- Identification of cheapest and most expensive flights  

---

## 📊 Key Insights  

- Business class tickets are significantly more expensive than Economy class  
- Ticket prices increase as the departure date approaches  
- Non-stop flights tend to be more expensive  
- Duration has a weak positive correlation with price  
- Certain airlines consistently have higher pricing

## 📂 Dataset Description  

| Column Name        | Description |
|-------------------|------------|
| airline           | Airline name |
| flight            | Flight code |
| source_city       | Source city |
| destination_city  | Destination city |
| departure_time    | Departure time |
| arrival_time      | Arrival time |
| stops             | Number of stops |
| class             | Flight class (Economy / Business) |
| duration          | Flight duration |
| days_left         | Days left before departure |
| price             | Ticket price |
